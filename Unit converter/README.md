# Unit Convertor
A simple unit convertor that takes in a number as input at shows the equal imperial and metric values.

## Features
- Converts:
  - Meters -> Feet, Feet -> Meters
  - Liters -> Gallons, Gallons -> Liters
  - Kilograms -> Pounds, Pounds -> Kilograms
- Dynamic DOM updates
- Reusable unit conversion function

## Built With
HTML
CSS
JavaScript
