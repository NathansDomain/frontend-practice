# Frontend Practice
A collection of small frontend projects built while learning HTML, CSS, and JavaScript. This repo contains experiments, mini-apps, and UI exercises focused on improving core frontend fundamentals.

## Purpose
This repository is used to practice:
- DOM manipulation
- Event handling
- Layout and styling (CSS)
- JavaScript fundamentals

## Projects of Note

### Password Generator
Generates two random passwords and allows click-to-copy functionality.

### City Homepage
Static homepage layout built with HTML and CSS.
