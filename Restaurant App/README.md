# Restaurant Order App
A simple vanilla JS restaurant order app. Users can browse items and add them to cart before checking out.

## Features
- Dynamic menu rendered from data
- Add / remove items from cart
- Quantity tracking per item
- Live cart total calculation
- Checkout modal with form validation
- Responsive layout (basic)

## Built With
- HTML, CSS, JavaScript

### What I learned
- Event delegation
- State managment with arrays
- Form handling and validation
